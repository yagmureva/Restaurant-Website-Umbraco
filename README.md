This project is a dynamic restaurant website built using **Umbraco CMS**.  
All content is managed through the Umbraco Backoffice, allowing full control over texts, images, and menu items without changing the code.

The goal of this project was to practice real-world CMS usage and build a clean, flexible restaurant website structure.

## Features
- Dynamic content management with Umbraco Backoffice  
- Editable food menu and page sections  
- Hero sections with background images  
- Clean Razor views and structured templates  

## Tech Stack
- Umbraco CMS  
- ASP.NET Razor  
- Bootstrap  
- HTML & CSS  

## What I Learned
- Working with Umbraco content models and templates  
- Managing dynamic content via Backoffice  
- Building reusable Razor views  
- Structuring a CMS-driven website
